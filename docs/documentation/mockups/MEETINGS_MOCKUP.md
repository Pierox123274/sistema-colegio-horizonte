# Mockup — Videoclases / reuniones

## Objetivo

Programar y unirse a sesiones sincrónicas.

## Usuarios

Docente (crear), estudiante (unirse).

## Componentes

`MeetingCard`, `UpcomingMeetingsPanel`, formulario con campo **pegar enlace** Meet/Zoom/Teams.

## Flujo

Crear → notificación → unirse → registro de asistencia virtual.
