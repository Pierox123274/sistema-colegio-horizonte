# Mockup — Tutor IA y copiloto

## Objetivo

Asistencia conversacional (estudiante) y generación asistida (docente).

## Usuarios

Estudiante, docente.

## Componentes

Chat con burbujas, panel coach (resumen, mini quiz), wizard de generación en copiloto.

## Estados

IA deshabilitada → aviso y plantillas locales; caché → indicador discreto.

## Seguridad UX

Sin datos de otros estudiantes; tono institucional.
