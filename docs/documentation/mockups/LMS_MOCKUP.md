# Mockup — LMS (aula virtual)

## Objetivo

Ver aula, tareas, exámenes y entregas.

## Usuarios

Docente (gestión), estudiante (consumo).

## Componentes

Listado de aulas, detalle con pestañas tareas/exámenes, formularios de entrega.

## Flujo visual

Docente: crear → publicar → calificar. Estudiante: pendiente → enviado → calificado.
