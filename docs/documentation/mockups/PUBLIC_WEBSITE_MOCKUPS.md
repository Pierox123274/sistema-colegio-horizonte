# Mockup — Sitio web público

## Objetivo

Presentar la institución, admisión, niveles, noticias y contacto.

## Usuarios

Visitantes, familias prospecto, comunidad educativa.

## Componentes

- `PublicNavbar`, hero con `PublicHeroImage`, secciones editoriales, footer institucional.  
- Páginas: Inicio, Nosotros, Niveles, Noticias, Admisión, Contacto.

## Navegación

Menú mega responsive; CTA hacia admisión y portal de acceso.

## Experiencia

Premium, animaciones suaves (Framer Motion), imágenes desde CMS.

## Estados

Carga skeleton en listados; vacío si no hay noticias.

## Responsive

Menú hamburguesa en móvil; grillas 1 columna.

## Mejoras futuras

Blog con categorías ampliadas; multiidioma (preparado a nivel contenido).
