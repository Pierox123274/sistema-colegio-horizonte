# Mockup — Pantalla de login

## Objetivo

Acceso seguro unificado a intranet y portales.

## Usuarios

Todos los roles internos.

## Componentes

Formulario email/contraseña, recordatorio institucional, enlace recuperación.

## Experiencia

Fondo institucional; validación inline; mensajes sin revelar si el usuario existe.

## Errores comunes

Credenciales incorrectas → mensaje genérico; cuenta desactivada → aviso administrativo.
