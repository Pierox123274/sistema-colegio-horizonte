# Guía de capturas de pantalla

## Carpeta sugerida

`docs/documentation/screenshots/images/`

## Resolución

- Escritorio: **1920×1080** o 1440×900.  
- Móvil (opcional): **390×844** (viewport iPhone).

## Nomenclatura

| Archivo | Pantalla |
|---------|----------|
| `public-home.png` | Inicio sitio público |
| `login.png` | Login |
| `admin-dashboard.png` | Dashboard intranet admin |
| `teacher-dashboard.png` | Dashboard docente |
| `student-dashboard.png` | Dashboard estudiante |
| `lms-classroom.png` | Detalle aula virtual |
| `cms-dashboard.png` | Listado CMS |
| `ai-tutor.png` | Tutor IA estudiante |
| `ai-copilot.png` | Copiloto docente |
| `gamification.png` | Gamificación estudiante |
| `meetings.png` | Listado reuniones |
| `integrations.png` | Panel integraciones |
| `ai-analytics.png` | Analítica IA admin |
| `notifications.png` | Centro notificaciones |

## Buenas prácticas

- Usuario de demo sin datos personales reales.  
- Modo claro y una captura dark mode opcional.  
- Recortar barras del SO si es para informe académico.
