# Manual — LMS

## Docente

1. Crear o abrir aula virtual.  
2. Publicar tarea con instrucciones y fecha.  
3. Calificar entregas con feedback.  
4. Crear examen online y revisar intentos.

## Estudiante

1. Entrar al aula desde menú Classrooms.  
2. Leer tarea y adjuntar entrega.  
3. Iniciar examen solo cuando esté habilitado.

## Integración

Calificaciones bajas pueden generar recomendaciones adaptativas automáticas.
