# Manual — Administrador

## Ingreso

1. Abrir URL institucional `/login`.  
2. Credenciales provistas por TI.  
3. Tras login → dashboard intranet.

## Funciones principales

- Usuarios y roles.  
- Estructura académica y años lectivos.  
- Reportes y analítica.  
- CMS del sitio web.  
- Seguridad y auditoría.  
- Salud del sistema, respaldos, integraciones.  
- IA institucional y gamificación (configuración).

## Buenas prácticas

- Crear usuarios con rol mínimo necesario.  
- Revisar `/intranet/system/health` semanalmente.  
- No compartir credenciales de administrador.

## Errores comunes

- “403 Forbidden” → rol incorrecto o sesión expirada.  
- IA no responde → verificar `AI_TUTOR_ENABLED` y claves en servidor (TI).
