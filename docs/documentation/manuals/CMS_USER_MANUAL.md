# Manual — CMS (sitio web)

## Usuarios

Administrador o editores con permiso CMS.

## Pasos típicos

1. Biblioteca de medios → subir imágenes.  
2. Noticias → crear artículo con editor enriquecido.  
3. Páginas → editar secciones del sitio.  
4. Menús y configuración → logo, datos de contacto.

## Recomendaciones

- Optimizar imágenes antes de subir.  
- Revisar vista pública en otra pestaña tras publicar.
