# Manual — Estudiante

## Ingreso

Login con usuario vinculado a ficha de estudiante.

## Funciones

- Ver notas y asistencia.  
- Entrar a aulas virtuales y entregar tareas.  
- Rendir exámenes online en la ventana indicada.  
- Usar tutor IA y coach (resumen, práctica, mini quiz).  
- Unirse a reuniones desde panel de meetings.  
- Ver progreso de gamificación.

## Buenas prácticas

- El tutor IA orienta; no reemplaza al docente.  
- Entregar tareas antes de la fecha límite.

## Errores comunes

- Portal vacío → contactar secretaría para vincular usuario con ficha.
