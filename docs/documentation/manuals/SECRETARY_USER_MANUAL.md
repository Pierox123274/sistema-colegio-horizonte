# Manual — Secretaría

## Ingreso

Login → intranet (dashboard administrativo acotado).

## Funciones

- Matrículas y datos de estudiantes/apoderados.  
- Registro de pagos y consulta de pensiones.  
- Asistencia.  
- Comunicados (según permisos).  
- Inventario y ventas (si aplica al rol).

## Restricciones

Sin acceso típico a: configuración CMS completa, integraciones, analítica IA admin, seguridad avanzada.

## Buenas prácticas

- Verificar año académico activo antes de matricular.
