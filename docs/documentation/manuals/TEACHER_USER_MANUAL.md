# Manual — Docente

## Ingreso

Login → portal `/teacher/dashboard`.

## Funciones

- Ver asignaciones y secciones del año activo.  
- Gestionar aulas virtuales (tareas, exámenes).  
- Crear diagnósticos y ver resultados.  
- Consultar riesgo académico de su alumnado.  
- Usar copiloto IA para generar materiales (revisar antes de publicar).  
- Programar videoclases pegando enlace Meet/Zoom/Teams.

## Recomendaciones

- Revisar siempre el contenido generado por IA antes de exportar al LMS o banco de preguntas.  
- Usar panel pedagógico como punto de partida diario.

## Errores comunes

- No poder crear reunión sin enlace → pegar URL manual de la videollamada.  
- No ver estudiante → verificar matrícula en su sección.
