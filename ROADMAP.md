# ROADMAP - Sistema Web Institucional e Intranet I.E.P. Horizonte

## Stack aprobado
- Laravel 11/12
- Inertia.js
- React
- TypeScript
- Tailwind CSS
- MySQL
- Laravel Sanctum
- Spatie Laravel Permission
- PHPUnit / Pest
- Cypress
- BDD con Gherkin
- PDF
- Boletas térmicas
- ISO 9001
- ISO/IEC 27001
- ISO/IEC 25010
- ISO/IEC 29119
